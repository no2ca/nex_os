# nex_os

Nonica's EXperimental Operating System

Currently WIP.

## Implemented Features

- Memory
    - Page allocator (bump allocator)
    - SV39 paging
- Process
    - User mode process
    - Round-robbin scheduler
    - Context switch (Struct Based)
    - ELF loader
- Trap
    - S-mode Trap Handler
    - SBI console output
